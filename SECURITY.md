# Security policy

This is a documentation repo — no compiled code, no app, no dependencies
to update. The realistic risks are:

- **Secrets, tokens, or live URLs** accidentally pasted into a doc or example.
- **Patterns recommended here** that, if followed literally, would weaken
  someone's actual production system.
- **Broken or hijacked outbound links** in the playbook.

If you find any of those, please email **backcountry@tronboll.us** instead
of opening a public issue. Use "ai-dev-governance security" in the subject
line. I'll respond within a few days and patch the docs.

For all other gaps, contradictions, or corrections, open an Issue or
Discussion — that's the normal feedback path.

## Supported versions

The `main` branch is the only supported version. There are no releases or
tags. If you're following a deep-dive, follow the version on `main`.
