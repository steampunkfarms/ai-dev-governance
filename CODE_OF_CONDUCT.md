# Code of Conduct

This repo is a playbook for running AI coding agents. It exists because real
problems shipped real bad code in production. The interactions here should
be the same shape: honest, specific, and focused on the work.

## What's expected

- **Argue patterns, not people.** "This rule misses case X" is welcome.
  "You're wrong because you're [adjective]" is not.
- **Be specific.** "This breaks for me" with no context isn't actionable.
  A repro, a file path, or the exact wall you hit makes a fix possible.
- **Assume your reader is busy.** Tight reports get acted on; sprawling
  ones get parked.
- **Disagreement is fine; harassment is not.** If you wouldn't say it to a
  colleague in front of their team lead, don't say it here.

## What's not OK

- Personal attacks, slurs, harassment, doxxing, sustained off-topic
  hostility, or trying to bait other readers into the same.
- Promoting tools or services by pretending to be a user asking a question.
- Repeated bad-faith argument after the maintainer has asked you to stop.

## Enforcement

This is a one-maintainer repo. I'll lock threads, delete comments, and
block accounts when I have to. I won't run a tribunal — if behavior
crosses the line, the consequence happens and the thread moves on.

For serious issues (off-platform harassment, threats, doxxing related to
activity here), email **backcountry@tronboll.us**.

## Scope

Applies to issues, PRs, Discussions, commit messages, and any space tied
to this repo. Doesn't apply to your other GitHub activity unless it spills
into this project.
